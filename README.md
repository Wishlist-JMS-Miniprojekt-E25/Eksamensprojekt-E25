# Projekt: CanWeMakeIT
Projektet CanWeMakeIT har til formål, at give projektledere, i primært software udviklings processer, et projektplanlægning og kalkulerings værktøj.  
  
Projektplanlægning kan hurtigt gå hen og blive meget rodet, da der er meget at holde styr på. CanWeMakeIT er værktøjet, der løser disse problemer. Det udtrykker klart og tydeligt de projekter man leder og om man er efter tidsplanen. Det giver altså et bedre overblik over projektets gang med specielt fokus på tidshorisonten gennem beregning af arbejdstimer.  

### Hvordan komme man i gang?  
CanWeMakeIT er programmeret i Java, derfor skal man have java 21 installeret på sit system, samt en vilkårlig IDE. Man skal også have mySQL installeret, så man kan køre diverse data scripts.  
  
Når en lokal database er oprettet, og mySQL scripts er kørt, så kan man bare køre programmet.  
  
Husk at konfigurere dine "Environment variables", så du er forbundet til din lokale database. Gå til "application.properties" og sæt spring.profiles.active=dev

### Hvordan bruger man værktøjet?  
CanWeMakeIT er et meget intuitivt planlægningsværktøj med en GUI, så det er nemt at navigere i. De forskellige funktionaliteter er mere eller mindre beskrevet 1 til 1 på de knapper man interagerer med.  

### Key feature:  
Når man afslutter projekter, bliver de arkiveret, og man kan gå ind og se dem på samme måde, som man kan se de aktive projekter. Det giver mulighed for analyse af hvorvidt deadlines blev overholdt eller om projektet blev færdig før tid.  
  
### Fremtidige tilføjelser til programmet:
til fremtidige tilføjelser, vil vi gerne gøre programmet mere medarbejder baseret. Dvs. mere potentiel information på medarbejder profilerne. Det er sådan noget som fx. en medarbejders tekniske færdigheder indenfor et givent område. Det åbner op for, at projekt lederen kan vælge sine medarbejdere nøje baseret på færdigheder, og dermed optimere tidshorisonten for et projekt.
